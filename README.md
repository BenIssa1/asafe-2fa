# Asafe2FA (PHP)

Petite lib **TOTP (RFC 6238)** compatible **Google Authenticator / Authy**.

## Installation (optionnel via Composer)

```bash
cd php
composer install
```

## Demo rapide (sans composer)

Depuis la racine du repo:

```bash
php php/demo.php
```

## Test RFC 6238 (sans phpunit)

```bash
php php/tests/rfc6238.php
```
